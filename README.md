# Theshan Geethanjana Portfolio — Professional Interactive Edition

A polished, responsive portfolio rebuilt with **React + TypeScript + Vite + Tailwind CSS + Framer Motion**.

## What is included

- Professional cinematic hero section
- Animated name reveal using scramble text
- Interactive fixed glassmorphism navigation bar
- Active section indicator in navbar
- Scroll progress line inside navbar
- Smooth animated mobile navigation panel
- Light and dark mode toggle button with saved preference
- Custom desktop cursor with hover and project-view states
- Mouse spotlight effect in hero and cards
- Magnetic call-to-action buttons
- 3D tilt effects on hero card and project cards
- Animated statistics counters
- Hover shine effect on buttons and navbar pills
- Interactive service cards with spotlight hover
- Skill stack filtering by category
- Animated skill cards with hover glow and icon motion
- Project category filters
- Quick-view project buttons
- Project detail modal with ESC close and backdrop close
- Detailed process/workflow section
- Journey timeline with hover motion
- Contact section with email copy button
- Fully responsive layout for desktop, tablet, and mobile
- Existing CV, project screenshots, GitHub, LinkedIn, and image assets preserved

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Run locally

```bash
npm install
npm run dev
```

Open:

```bash
http://localhost:5173
```

## Build for production

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Main editable files

```text
src/App.tsx       # Main portfolio content and interaction logic
src/index.css     # Full visual design, responsiveness, animations, light/dark theme
public/           # Images, icons, screenshots, and CV
```

## Notes

The light/dark theme is stored in `localStorage`, so the selected mode remains after refresh.
